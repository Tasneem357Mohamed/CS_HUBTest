# Introduction to the Project
This project is a Django-based application that provides a set of utility functions for managing user subscriptions, sending staff alerts, and generating profile paths.

## Project Architecture
The project is structured as follows:
- utils.py: contains utility functions for the application

## Installation
To install the project, follow these steps:
1. Clone the repository
2. Install the required dependencies
3. Run the application

## Usage
The project provides several utility functions, including:
- is_subscription_active: checks if a user's subscription is active
- get_formatted_total: formats the total order amount as a string
- send_staff_alert: sends an automated alert email to staff
- generate_profile_path: generates a unique filename for user uploads
- deactivate_inactive_users: deactivates users who haven't logged in for a year
